Инструкция по запуску сервера и тестировкой бэкенда заметок.
1) Склонировать репозиторий и подгрузить папку в интерпритатор python.
2) Находясь в интерпритаторе python и склонированой папке, заходим в терминал и вводим последовательно команды:
# Создайте виртуальное окружение
python -m venv venv
# Активируйте виртуальное окружение
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
# Установите зависимости из requirements.txt
pip install -r requirements.txt
# Запускаем сервер
python -m main
3) В терминале ищем ссылку на сайт с сервером
http://127.0.0.1:8000
4) Для тестировки добавляем /docs
http://127.0.0.1:8000/docs

# Для остановки сервера используем
Ctrl + C
